# prj014_Aip_HHW
Github repository for Hatch-West et al. in prep 

This directory contains the following: 

- Transcriptomics processing scripts for HPC
- 16S HPC processing script (dada2.r)
- rmd scripts for QC and statistical analysis for qPCR, 16S and transcriptomics results

It also has non tracked repositories for output and data folders related to project
